
# CIDR

10.0.0.0/24
- Each number is an otic, 8 bits
0.0.0.0 
8+8+8+8 = 32 bits total

The last number says what is locked.

So with 24 after the slash, we are using 24/32. We can only change the last number

10.0.0.0/16
- We can change last 2 numbers

10.0.0.0/8
- We can change last 3 numbers

/24  10.0.0.1 - 10.0.0.254
/16 10.0.0.1 - 10.0.255.254
/8 10.0.0.1 - 10.255.255.254
